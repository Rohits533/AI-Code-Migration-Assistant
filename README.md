# AI-Code-Migration-Assistant
# 🤖 AI Code Migration Assistant

An AI-powered tool that automatically migrates and upgrades Python code using AST (Abstract Syntax Tree) transformation and Groq AI.

---

## 🚀 Features

- **AST-Based Transformation** – Rename variables, arguments, and apply custom migrations.
- **AI-Powered Explanations** – Understand why code changed with AI-generated summaries.
- **Diff View** – See exactly what changed.
- **Batch Processing** – Migrate entire files or projects.
- **Streamlit UI** – Clean, dark-themed interface.

---

## 🛠️ Tech Stack

- Python
- AST (Abstract Syntax Tree)
- Streamlit
- Groq API
- `astunparse`
